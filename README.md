## Requirement

To this project your need to have:

- Nodejs v18 [Nodejs](https://nodejs.org/en/)
- docker [Docker](https://www.docker.com/)

In this project your need create a file **.env** and put your config database.

```yml
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_NAME=
DB_ADMIN_EMAIL=
```
